# 개선점
- 4진법으로 가능한 방향 조합 관리
- 4진법으로 정해준 각 CCTV의 방향에 대해서, 포함된 진출 방향 수 만큼 함수 호출
  - 이 때, 함수 내에서 전달받은 진출 방향은 `% 4`로 관리
- 예: 2번 CCTV
```cpp
// d는 4진법에 의해 정해짐 (0 ~ 3)
fill_line(x, y, d);
fill_line(x, y, d + 2);

void fill_line(int x, int y, int d) {
  // 가능한 d의 조합: (0, 2), (1, 3), (2, 4), (3, 5)
  d %= 4;
  // 변환된 d의 조합: (0, 2), (1, 3), (2, 0), (3, 1)
  ...
}
```
